# Comunicador celular G16T

¡IMPORTANTE!

## G16T_3xxx - la 3ra revisión de HW en G16T. Cambios:

1.  Soporte de tarjeta SIM para tarjeta SIM 4F (tamaño Nano).

2.  Realizado el cableado de 4 hilos del G16T al comunicador de línea telefónica (TLC) en el panel de alarma para recepción de informes de Contact ID y supervisión de línea telefónica.

3.  2 entradas IN de tipo NC (configuración predeterminada).

4.  1 salida OUT de tipo OC, 150 mA.
