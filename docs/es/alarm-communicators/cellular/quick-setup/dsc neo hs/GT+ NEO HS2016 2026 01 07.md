# DSC NEO HS2016 con GT/GT+/GET configuración rápida

Pasos breves de cableado y programación para conectar el comunicador GT/GT+/GET al panel DSC NEO HS2016 usando terminales TIP/RING, y luego añadir el sistema a Protegus2. Use esto junto con los manuales completos para otros ajustes. (Las etiquetas de terminal pueden variar ligeramente entre GT/GT+/GET, pero las conexiones son las mismas.)

!!! caution "Precaución"
    La instalación y el servicio deben ser realizados solo por personal cualificado. Desconecte la alimentación antes de cablear. Los cambios no autorizados anulan la garantía.

## Requisitos

1. Firmware GT/GT+/GET 1.21, SIM insertada, PIN deshabilitado, plan de datos activo.
1. Panel DSC NEO HS2016 con acceso al teclado (código de instalador disponible).
1. Número de cuenta CMS si reporta a CMS.
1. Cuenta de empresa/instalador de Protegus2 y IMEI del comunicador.

## Cableado

Siga el esquema de abajo para conectar el comunicador al panel:

| **Terminal GT/GT+/GET** | **Panel DSC** | **Notas**                             |
| ----------------------- | ------------- | ------------------------------------- |
| +12V DC/-12V DC         | +AUX/-AUX     | Alimentación del comunicador          |
| TIP/RING                | TIP/RING      | Para emulación de línea PSTN (Contact ID) |
| 1 I/O                   | Z1            |                                       |

## Programación del panel de alarma DSC NEO HS2016 mediante el teclado

Usando el teclado del panel, entre en estas secciones y configúrelas como se indica:

**Habilitar reporte Contact ID**

| **Entrada de teclado** | **Descripción de la acción**                               |
| ---------------------- | ----------------------------------------------------------- |
| *85555                 | Entrar en modo de programación                              |
| 001*                   | Entrar en programación de “Zone type”                       |
| 066##                  | Configurar el tipo de zona “Momentary keyswitch”           |
| 301*                   | Entrar en programación de “Phone Number”                    |
| <1234##                | Introducir número telefónico (1234)                         |
| 310*                   | Entrar en programación de “Account Codes”                   |
| 1111                   | Introducir “System Account Code” (1111)                     |
| 001                    |                                                             |
| 2222#                  | Introducir “Partition 1 Account Code” (2222)                |
| 311                    | Entrar en programación de “Partition 1 Call Direction”      |
| 003*##                 | Habilitar “Partition Opening/Closing Call Direction” (Y)    |
| 350*                   | Entrar en programación de “Communicator Formats”            |
| 03#                    | 03 – Formato Contact ID.                                    |
| ##                     | Salir del modo de programación.                             |

!!! tip "Consejo"
    Después de completar la configuración y la instalación realice una comprobación del sistema:

    1. Cree un evento:

       - armando/desarmando el sistema con el teclado del panel de control.
       - provocando una alarma de zona cuando el sistema de seguridad está armado.

    2. Asegúrese de que el evento llegue al CMS (Central Monitoring Station) y a la aplicación Protegus2.
