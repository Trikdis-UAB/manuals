# Универсальная охранная панель с GT/GT+/GET быстрая настройка

Краткие шаги для подключения коммуникатора GT/GT+/GET к охранной панели, которая передает сообщения через PSTN телефонный дозвонщик, приема событий Contact ID через `TIP` / `RING` и управления постановкой/снятием из Protegus2 через зону ключа панели. Используйте это руководство вместе с полным руководством коммуникатора и руководством по программированию панели.

!!! caution "Осторожно"
    Установку и обслуживание должны выполнять только квалифицированные специалисты. Перед подключением отключите питание. Несанкционированные изменения аннулируют гарантию.

## Требования

- Прошивка GT/GT+/GET 1.21, SIM-карта установлена, PIN отключен, активен тариф с передачей данных.
- Охранная панель с PSTN телефонным дозвонщиком, поддерживающим Contact ID через DTMF тоны.
- Учетная запись компании/установщика Protegus2 и IMEI коммуникатора.
- Код установщика панели и руководство по программированию панели нужны только если панель еще не выполняет дозвон через телефонный дозвонщик, если нужно изменить настройки дозвонщика или если будет добавлена удаленная постановка/снятие через зону ключа.
- Свободная зона панели, которую можно запрограммировать как зону ключа, нужна только если требуется удаленная постановка/снятие.

!!! note "Примечание"
    Если панель ранее передавала события через телефонный дозвонщик, оставьте дозвонщик панели включенным и подключите коммуникатор к клеммам `TIP` / `RING` панели. Коммуникатор отвечает на вызов панели и автоматически принимает события Contact ID с любым ID учетной записи, переданным панелью. Изменяйте программирование панели только если панель еще не выполняет дозвон, настройки дозвонщика неправильные, ID учетной записи должен соответствовать требованиям ARC/CMS или нужно добавить зону ключа для удаленной постановки/снятия.

## Подключение

Перед подключением отключите питание панели и коммуникатора. Подключите коммуникатор к питанию панели, `TIP` / `RING` и зоне ключа, как показано ниже.

| Клемма GT/GT+/GET | Клемма охранной панели | Назначение |
| --- | --- | --- |
| `+12V` / `GND` | `AUX+` / `AUX-` | Питание коммуникатора от вспомогательного выхода панели. |
| `TIP` / `RING` | `TIP` / `RING` | Прием событий Contact ID от телефонного дозвонщика панели. |
| `OUT` / `I/O`, настроенный как PGM | Вход зоны ключа | Команды постановки/снятия из Protegus2. |
| `GND` / `COM` | Общая клемма зоны, если требуется | Общая точка для подключения зоны ключа. |

![Универсальная схема подключения TIP/RING и зоны ключа](../../../../../images/quick-setup/generic-dial-capture.svg)

!!! warning "Предупреждение"
    Подключайте зону ключа точно так, как указано в руководстве панели. Некоторые панели используют нормально открытый вход, нормально закрытый вход, EOL резистор, двойной EOL резистор или отдельную общую клемму.

## Программирование охранной панели

Пропустите этот раздел, если панель уже выполняет дозвон через телефонный дозвонщик и удаленная постановка/снятие не требуется. Коды программирования отличаются у каждого производителя и модели. Используйте руководство по программированию охранной панели только для тех пунктов, которые нужно изменить.

1. Включите PSTN телефонный дозвонщик панели.
2. Выберите тоновый / DTMF набор.
3. Выберите формат передачи Contact ID.
4. Укажите телефонный номер приемника. Если панель уже передавала события на ПЦН через телефонную линию, существующий номер обычно можно оставить без изменений. Для новой установки dial-capture введите любой номер приемника длиннее 4 цифр, если руководство панели не требует другой формат.
5. Если требуется передача на ARC/CMS, укажите ID учетной записи панели, предоставленный ПЦН. Коммуникатор может передавать ID учетной записи, отправленный панелью, поэтому отдельный Object ID коммуникатора не требуется для базового приема через дозвонщик.
6. Включите события, которые должны передаваться: тревоги, восстановления, неисправности, тамперы и события постановки/снятия, когда они требуются.
7. Запрограммируйте зону, подключенную к выходу коммуникатора, как зону ключа только если требуется удаленная постановка/снятие из Protegus2.
8. Выберите тип зоны ключа, соответствующий режиму выхода коммуникатора: моментный / импульс или удерживаемый / уровень.
9. Назначьте зону ключа правильному разделу или области, затем сохраните настройки и выйдите из программирования.

!!! important "Важно"
    Удаленная постановка/снятие из Protegus2 работает только когда подключенная зона панели запрограммирована как зона ключа. Статус в приложении также зависит от передачи панелью событий постановки и снятия через дозвонщик.

## Проверка системы

- [ ] Поставьте и снимите панель с охраны с клавиатуры.
- [ ] Создайте тестовую тревогу, когда система поставлена на охрану.
- [ ] Убедитесь, что события поступают в Protegus2. Если используется передача на ARC/CMS, убедитесь, что ПЦН получает события с ожидаемым ID учетной записи.
- [ ] Поставьте и снимите систему с охраны из Protegus2, если зона ключа подключена.
- [ ] Убедитесь, что панель выполняет команду приложения, а затем передает событие постановки/снятия обратно в Protegus2.
- [ ] Если события не поступают, проверьте включение дозвонщика панели, DTMF набор, формат Contact ID, телефонный номер, подключение `TIP` / `RING` и ID учетной записи только если ожидается передача на ARC/CMS.
