??? question "Как настроить один считыватель Wiegand (без клавиатуры) для подачи импульса на выход, управляющий дверным замком?"

<!-- --8<-- [start:sp3-wiegand-reader-door-output-body] -->
    1. Подключите линии данных считывателя к клеммам **GRN**, **YEL** охранной панели.
    2. В TrikdisConfig установите **Тип клавиатуры = считыватель Wiegand**, даже если физическая клавиатура не подключена (окно «Модули» → закладка «Клавиатуры» → группа «Параметры клавиатуры»).
    3. Нужному выходу установите **Описание выхода = Удаленное управление** и **Время импульса, с** (например, 5) в окне **«PGM выходы»**.
    4. На закладке **«Управление»** окна **«PGM выходы»** отметьте **Вкл.** для этого считывателя, укажите в поле **PGM** номер выхода и установите **PGM режим = Импульс**.

    !!! note
        Карточки/брелоки RFID (Wiegand) нельзя зарегистрировать простым прикладыванием к считывателю — введите Код iButton или RFID брелока каждой карточки вручную, в виде **десятичного** числа, в разделе «Регистрация RFID карточек (брелоков)».
