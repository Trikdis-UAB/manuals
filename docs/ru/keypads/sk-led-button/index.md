> [!NOTE]
> Клавиатура “SK-LED button” для управления охранной сигнализацией отображает состояние 16 зон и 2 разделов. Также клавиатура “SK-LED button” может быть назначена для управления одной или двумя выбранными зонами (версия прошивки клавиатуры от FW:1.0, версия прошивки панели управления от FW:SP3_xxx4_0120). Клавиатура отображает статус выбранного раздела и его зон.

## Постановка/Снятие с охраны

### Постановка на охрану (ARM)

1. Убедитесь, что нет активных зон (ни один индикатор зон не горит).

2. Нажмите кнопку <img src="./image2.png" alt="" style="width:0.3500in;" />.

3. Введите код пользователя.

4. Нажмите номера разделов, которыми хотите управлять.

5. Нажмите кнопку <img src="./image3.png" style="width:0.3500in;" />.

6. Выйдите из помещения в течение времени задержки на выход.

После ввода кода индикатор <img src="./image2.png" alt="" style="width:0.3500in;" /> будет мигать в течение времени задержки на выход. Когда система поставлена на охрану, загорится индикатор <img src="./image2.png" alt="" style="width:0.3500in;" />.

### Снятие с охраны (DISARM)

1. Введите код пользователя.

2. Нажмите кнопки с номерами разделов, которые нужно снять с охраны.

3. Нажмите кнопку <img src="./image3.png" style="width:0.3500in;" />.

После снятия сигнализации с охраны должен загореться индикатор <img src="./image4.png" alt="" style="width:0.3500in;" />.

### Включение режима SLEEP

(Охраняется периметр помещения. Внутри разрешено движение. Если открыть входную дверь, тревога сработает немедленно):

1. Нажмите кнопку <img src="./image4.png" style="width:0.3500in;" />.

2. Введите код пользователя.

3. Нажмите кнопку раздела для включения.

4. Нажмите кнопку <img src="./image3.png" style="width:0.3500in;" />.

5. Режим SLEEP включится сразу, без задержки выхода.

Индикатор <img src="./image4.png" alt="" style="width:0.3500in;" /> загорится, когда режим SLEEP включен.

### Включение режима STAY

(Охраняется периметр помещения. Внутри разрешено движение. Открытие входной двери включает задержку, в течение которой нужно снять систему с охраны):

1. Нажмите кнопку <img src="./image6.png" style="width:0.3500in;" />.

2. Введите код пользователя.

3. Нажмите кнопку раздела для включения.

4. Нажмите кнопку <img src="./image3.png" style="width:0.3500in;" />.

5. Режим STAY включится сразу, без задержки выхода.

Индикатор <img src="./image7.png" alt="" style="width:0.3500in;" /> загорится, когда режим STAY включен.

> [!NOTE]
> Для изменения статуса раздела охранной сигнализации на противоположный, введите код пользователя и выберите нужный раздел. Чтобы удалить введенные символы или команду, нажмите <img src="./image8.png" alt="" style="width:0.3500in;" />.

## Кнопки помощи

**<u>Отправление экстренного сообщения охранному предприятию о возникшей опасности</u>**
- **Паника** – нажмите и удерживайте кнопку **1** 3 сек..

- **Медицина** – нажмите и удерживайте кнопку **4** 3 сек..

- **Пожар** – нажмите и удерживайте кнопку **7** 3 сек..

## Регулировка яркости экрана и громкости индикации

### Регулировка яркости подсветки кнопок клавиатуры

1. Нажмите кнопку <img src="./image9.png" style="width:0.3500in;" />, затем **1**.

2. Выберите желаемую яркость подсветки кнопок клавиатуры с помощью кнопок **2** и **8**.

3. Нажмите <img src="./image3.png" style="width:0.3500in;" />, чтобы сохранить новое значение. Нажмите кнопку <img src="./image8.png" style="width:0.3500in;" /> для выхода.

**<u>Регулировка яркости подсветки индикации зон:</u>**

4. Нажмите кнопку <img src="./image9.png" style="width:0.3500in;" />, затем **4**.

5. С помощью кнопок **2** и **8** выберите желаемую яркость подсветки индикации зон.

6. Нажмите <img src="./image3.png" style="width:0.3500in;" />, чтобы сохранить новое значение. Нажмите кнопку <img src="./image8.png" style="width:0.3500in;" /> для выхода.

**<u>Регулировка громкости нажатия клавиш клавиатуры:</u>**

7. Нажмите кнопку <img src="./image9.png" style="width:0.3500in;" />, затем **7**.

8. Выберите предпочтительный уровень звуковой индикации с помощью кнопок **2** и **8**.

9. Нажмите <img src="./image3.png" style="width:0.3500in;" />, чтобы сохранить новое значение. Нажмите кнопку <img src="./image8.png" style="width:0.3500in;" /> для выхода.
> [!NOTE]
> Отключение подсветки клавиатуры в режиме ожидания. Нажмите <img src="./image9.png" style="width:0.3500in;" /> с последующим нажатием **5** переключаем состояния подсветки индикаторов клавиатуры в режиме ожидания. Нажмите <img src="./image3.png" style="width:0.3500in;" />, чтобы сохранить новое значение. Нажмите кнопку <img src="./image8.png" style="width:0.3500in;" /> для выхода.

## Ввод или изменение кодов пользователей или мастер-кода

### Ввод или изменение кодов пользователей

1. Нажмите кнопку <img src="./image9.png" style="width:0.3500in;" />, затем **0**.

2. Введите Мастер код (заводской код: 1234).

3. Введите требуемый двузначный порядковый номер пользователя, например, **02**.
> [!NOTE]
> Порядковый номер Мастер-кода – **01**.

4. Дважды введите новый код пользователя.

5. Введите номера разделов охранной сигнализации, которыми будет управлять пользователь.

6. Нажмите кнопку <img src="./image3.png" style="width:0.3500in;" />. Нажмите кнопку <img src="./image8.png" style="width:0.3500in;" /> для выхода.

## Перезапуск пожарных (дымовых) извещателей

### Для перезапуска пожарных (дымовых) извещателей необходимо

- Нажмите и удерживайте кнопку <img src="./image8.png" style="width:0.3500in;" /> в течение 3 сек.
> [!NOTE]
> После активации пожарного извещателя, он автоматически не перезапускается. Пожарный извещатель перезапускается только вручную.

## Временное отключение мониторинга зоны (функция BYPASS)

### Включение функции BYPASS

1. Нажмите кнопку <img src="./image9.png" style="width:0.3500in;" />, затем **6**.

2. Введите код пользователя. Индикатор <img src="./image10.png" style="width:0.3500in;" /> начнет мигать зеленым.

3. Введите двухзначные номера зон, которые хотите отключить.

4. Нажмите кнопку <img src="./image3.png" style="width:0.3500in;" />. Индикатор <img src="./image10.png" style="width:0.3500in;" /> начнет светиться зеленым.

**<u>Отключение функции BYPASS:</u>**

Повторите те же шаги, что и для включения функции **Bypass**.

## Удаление кодов пользователей

### Удаление кода пользователя

1. Нажмите кнопку <img src="./image9.png" style="width:0.3500in;" />, затем **0**.

2. Введите Мастер код.

3. Введите требуемый двузначный порядковый номер пользователя, например, **02**.

4. Нажмите кнопку <img src="./image5.png" style="width:0.3500in;" />. Раздастся звуковой сигнал и кнопка, указывающая порядковый номер пользователя, код которого был удален из памяти, погаснет.

5. Нажмите кнопку <img src="./image3.png" style="width:0.3500in;" />. Нажмите кнопку <img src="./image8.png" style="width:0.3500in;" /> для выхода.  Кроме того, клавиатуру „SK- LED button“ можно назначить для управления одной или несколькими желаемыми разделами охранной сигнализации (версия прошивки клавиатуры от FW:1.0, версия прошивки охранной панели от FW:SP3_xxx4_0120).
