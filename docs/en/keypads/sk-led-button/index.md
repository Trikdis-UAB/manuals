## Alarm system arming / disarming

## <u>Alarm system ARM:</u>

1.  Make sure all zones are not violated (none of zone indicators are illuminated).

2.  Press the [<img alt="" src="./image2.png" style="width:0.124016in;height:0.141732in" />] button.

3.  Enter User code.

4.  Select the button of area to be activated.

5.  Press [<img alt="" src="./image3.png" style="width:0.180000in;height:0.175000in" />] button.

6.  Exit the premises within time delayed.

Once the code is entered the <img alt="" src="./image2.png" style="width:0.124016in;height:0.141732in" /> indicator will flash for duration of exit time delay. When the system is armed, the <img alt="" src="./image2.png" style="width:0.124016in;height:0.141732in" /> indicator will light up.

## <u>SLEEP mode activation:</u>

(Premises perimeter is secured. Movement inside is allowed. If you open the entrance door, the alarm will actuate immediately):

1.  Press [<img alt="" src="./image4.png" style="width:0.153543in;height:0.141732in" />] button.

2.  Enter User code.

3.  Press the button of area to be activated.

4.  Press [<img alt="" src="./image3.png" style="width:0.180000in;height:0.175000in" />] button.

5.  SLEEP mode will turn on immediately, without exit delay.

The <img alt="" src="./image4.png" style="width:0.153543in;height:0.141732in" /> indicator will illuminate when SLEEP mode is on.

## <u>Alarm system DISARM</u>:

1.  Enter User code.

2.  Press the button of particular area you wish to disarm.

3.  Press [<img alt="" src="./image3.png" style="width:0.180000in;height:0.175000in" />] button.

When the alarm is disarmed the <img alt="" src="./image5.png" style="width:0.147638in;height:0.141732in" /> indicator must be illuminated.

## <u>STAY mode activation</u>:

(Premises perimeter is secured. Movement inside is allowed. Any opening of entrance door enables time delay during which you have to disarm the alarm):

1.  Press [<img alt="" src="./image6.png" style="width:0.165354in;height:0.165354in" />] button.

2.  Enter User code.

3.  Press the button of area to be activated.

4.  Press [<img alt="" src="./image3.png" style="width:0.180000in;height:0.175000in" />] button.

5.  STAY mode will turn on immediately, without exit delay.

The <img alt="" src="./image7.png" style="width:0.141732in;height:0.141732in" /> indicator will illuminate when STAY mode is on.

## Emergency buttons

#### Fire (smoke) detector reset

**<u>To send emergency message to your security service:</u>**

- **Police** – hold [**1**] button pressed for 3 sec.

- **Medical Aid** – hold [**4**] button pressed for 3 sec.

- **Fire Service** – hold [**7**] button pressed for 3 sec.

**<u>To reset fire (smoke) detectors:</u>**

- Hold button [<img alt="" src="./image8.png" style="width:0.205000in;height:0.140000in" />] pressed for 3 seconds.

**<u>Note.</u>** Fire (smoke) detectors do not reset automatically after fire emergency detection. They can be reset only manually.

## Illumination brightness and indication sound adjustment

#### Temporary zone monitoring deactivation (BYPASS function)

## <u>Brightness of the keypad buttons</u>

1.  Select the preferred brightness of the keypad button backlight using the [**2**] and [**8**] buttons.

**<u>The brightness of the zone LED indication</u>**

1.  Select the preferred illumination brightness of zone indication using [**2**] and [**8**] buttons.

**<u>Keypad keystroke volume</u>**

1.  Select the preferred sound indication level using [**2**] and [**8**] buttons.

**<u>Note:</u>** Turn off keypad lighting in standby mode. Pressing [<img alt="" src="./image9.png" style="width:0.171260in;height:0.177165in" />] followed by [**5**] toggles the backlight states of the keypad indicators in standby mode. Press [<img alt="" src="./image3.png" style="width:0.180000in;height:0.175000in" />] to save the new value. Press the [<img alt="" src="./image8.png" style="width:0.205000in;height:0.140000in" />] button to exit.

**<u>BYPASS function activation:</u>**

2. Enter the alarm control code. The [<img alt="" src="./image10.png" style="width:0.171260in;height:0.153543in" />] indicator starts flashing green.

**<u>BYPASS function deactivation:</u>**

Repeat the same actions as in deactivation of particular zone monitoring.

## Entering or changing / User or Master codes

#### Deleting User codes

**<u>To enter a new or change the existing User code:</u>**

1.  Enter *Master* code, default code – 1234.

**<u>Note.</u>** Sequence number of *Master* code - [**01**].

**<u>To delete User code:</u>**

1.  Press the sequence numbers of areas which should be controlled by User.
1.  Enter *Master* code.

The keypad “SK-LED button” for alarm system control displays the states of 16 zones and 2 partitions. Also, the “SK-LED button” keypad can be assigned to control one or two desired areas (keypad firmware version from FW:1.0, control panel firmware version from FW:SP3_xxx4_0120). The keypad will display the statuses of the assigned area and area zones.
