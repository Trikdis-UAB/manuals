# GSM komunikatorius G16T


1.  SIM card holder for 4F SIM card (Nano size).

2.  Implemented 4-wire wiring of G16T to telephone line communicator (TLC) on security alarm panel for both Contact ID report receiving and Telephone Line Supervision.

3.  2 inputs IN of NC type (default settings).

4.  1 output OUT of OC type, 150 mA.

| Clamp | Description |
|----|----|
| +DC | 10 - 18 VDC power supply |
| -DC | Common (negative) |
| TIP | For connecting to the TIP clamp of security control panel (receiving of Contact ID reports) |
| RING | For connecting to the RING clamp of security control panel (receiving of Contact ID reports) |
| T-1 /​ IN1 | IN1 of NC type (by default) or /​ For connecting to T-1 clamp on security control panel (for tel. line supervision) |
| R-1 /​ IN2 | IN2 of NC type (by default) or /​ For connecting to R-1 clamp on security control panel (for tel. line supervision) |
| OUT | open-collector output |
| COM | Common (negative) |
| A 485 | RS-485 bus, positive contact (for connecting expanders of iO series) |
| B 485 | RS-485 bus, negative contact (for connecting expanders of iO series) |

| Laidinių sujungimų schema nepakeitus numatytųjų nustatymų (2-laidis G16T prijungimas prie centralės tel. komunikatoriaus). Wiring if clamps set by default (2-wire TLC and G16T wiring). |
