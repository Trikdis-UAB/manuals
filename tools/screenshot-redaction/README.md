# Screenshot Redaction Tools

This directory stores redaction workflow specifications and profile notes that are implementation-oriented and reused across manuals.

## Profiles

- IPcom5 profile: `tools/screenshot-redaction/ipcom5/`

## Why this lives under `tools/`

- Easier to discover from repo root.
- Keeps process/tooling documentation separate from published manual content.
- Allows adding additional product profiles without changing project-level manual structure.
